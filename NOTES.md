# PINN Project — Notes

## Що це

Physics-Informed Neural Network для 1D рівняння теплопровідності з рухомим пальником.

Задача: є металева труба довжиною l, по ній рухається пальник зі швидкістю v.
Знайти розподіл температури ΔT(x, t) для будь-якого матеріалу та швидкості.

Рівняння:
```
∂ΔT/∂t = α · ∂²ΔT/∂x² + i_eff · δ(x − x₀ − v·t)
```
де α — теплодифузійність матеріалу, i_eff — потужність пальника.

## Архітектура мережі

**Вхід:** (x_norm, t_norm, α_n, l_n, i_eff_n, x₀_n, v_n) — 7 чисел
**Вихід:** ΔT — підйом температури в точці x в момент t

**Pipeline:**
```
(x, t) → FourierFeatures(σ) → 128 ознак
                                        ┐
(α, l, i_eff, x₀, v) ─────────────────→ cat → [133] → MLP(4×256, tanh) → × t_norm → ΔT
```

`× t_norm` — hard IC: гарантує ΔT(x,0)=0 математично, без штрафного терму.

Параметри: ~232k ваг.

---

## Три ключові покращення (вже реалізовані)

### 1. Hard Initial Condition
`output = t_norm * MLP(features)` → при t=0 вихід завжди 0.
До цього: мережа порушувала IC, що "псувало" всю еволюцію в часі.

### 2. Adaptive Fourier Sigma (curriculum)
`σ` росте з 1.0 до 3.5 лінійно протягом навчання.
Спочатку мережа вчить плавні глобальні патерни, поступово деталізує гострий пік пальника.

### 3. Causal PDE Loss (Wang et al., JMLR 2024)
Ділимо час на 10 вікон. Вікно k отримує вагу:
```
w_k = exp(−ε · Σ_{j<k} L_j)
```
Поки рання динаміка (t малий) погана — пізні вікна майже вимкнені.
Мережа мусить вивчити початок перш ніж переходити до продовження.

**Що видалено:** L-BFGS фаза (підвищувала помилку з 186% → 289%).

## Поточні результати

Тренування на Vast.ai RTX 5060 Ti, 80 000 кроків Adam.

| Checkpoint | Крок | Validation error |
|---|---|---|
| model_30000.pt | 30 000 | **76%** ← найкращий |
| model_final.pt | 80 000 | 85.7% |

Форма кривої температури правильна. Проблема — систематична помилка в амплітуді (мережа недооцінює максимум).

До всіх покращень помилка була ~440%.

## Плани

### Наступний крок: PirateNets (JMLR 2024)
Прогресивна архітектура де кожен шар має адаптивний residual gate, ініціалізований біля нуля.
На початку навчання мережа поводиться як неглибока → поступово "розблоковує" глибину.
Вирішує патологічний градієнт у глибоких PINN на довгих часових горизонтах.

Паралельно: RAD sampling (Residual Adaptive Refinement) — більше точок там де residual найбільший.

### Далі: інверсна задача
Дано показання 2–3 датчиків → знайти матеріал (α) і параметри пальника (i_eff, x₀, v).
Деталі математики: `memory/problems/goal.md`.
